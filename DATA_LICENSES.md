# Data and output licenses

## Repository code

Source code in `src/`, `transportability/`, tests, and workflow files is
released under the MIT License in `LICENSE`.

## Project-authored materials

Original documentation, manuscript text, and figures in this repository are
released under the Creative Commons Attribution 4.0 International license
(CC BY 4.0), unless a file states otherwise.

## OpenNeuro data

OpenNeuro `ds004504` v1.0.9 and `ds006036` v1.0.6 are released under CC0. Raw
EEG is not redistributed. Users should download the identified snapshots and
cite their dataset DOIs and descriptors.

## P-ADIC Dryad data

P-ADIC DOI `10.5061/dryad.8gtht76pw` is released under Dryad's CC0 dedication.
Raw MATLAB files are not redistributed. Scholarly citation of the dataset and
associated article is expected.

## Derived outputs

Tracked predictions and feature tables use coded source participant identifiers
and contain no names or contact information. Reuse should preserve source
dataset citations and the claim boundaries documented in this repository.
